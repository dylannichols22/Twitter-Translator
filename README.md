# Speak a lil' Chinese for 'em

Firefox extension that translates Chinese Twitter/X content using the Anthropic (Claude) API.

## Features
- Translate tweet threads on Twitter/X
- Side panel view on desktop and full-screen overlay on Android
- Optional breakdowns (pinyin/gloss/notes)
- Save cards for later review

## Setup
1. Install the extension
2. Open the popup and add your Anthropic API key
3. Open a tweet thread and use the panel or translate view

## Data Use
Tweet content is sent to Anthropic for translation. See `PRIVACY.md` for details.
